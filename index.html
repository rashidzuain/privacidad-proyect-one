1. Información recopilada

La aplicación puede recopilar datos como nombre, número de teléfono, mensajes intercambiados, y datos de uso.

La información se utiliza para mejorar la funcionalidad y permitir la comunicación vía WhatsApp.

Los datos se almacenan de forma segura y no se comparten con terceros sin consentimiento.

Los usuarios pueden solicitar acceso, eliminación o corrección de sus datos, y revocar el consentimiento.

5. Seguridad

Aplicamos medidas para proteger los datos contra accesos no autorizados.
